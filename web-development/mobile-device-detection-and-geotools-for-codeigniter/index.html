Mobile Device Detection and Geotools for CodeIgniter

Recently I discovered GetSparks.org a Ruby Gem like package manager for the PHP framework, CodeIgniter. CodeIgniter is a great, lightweight framework and the sparks system makes setting up a new application even easier, providing lots of useful functionality, ready to work straight out of the box.

I’ve released a couple of sparks over the last few days.

  • mobiledetection – a port of the Coldfusion mobile device detection library I had previously released.
  • geotools – a (albeit small at the minute) collection of tools for performing geography related calculations and geocoding

If you’re not interested in using sparks (and I think you should, its well worth a look if you’re building CodeIgniter based applications) then you grab the code from my GitHub account where I’ve included manual installation and usage instructions.
